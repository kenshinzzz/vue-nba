## This is why we play
在了解React的时候找到了几个不错的应用，顺手用Vue实现了一下。

React Native : [react-native-nba-app](https://github.com/wwayne/react-native-nba-app)

React和Vue很相似，生命周期、路由管理、状态管理、组件、页面模板等。用Vue实现时代码几乎可以原封不动的实现。

从开发体验上，还是比较喜欢Vue。
